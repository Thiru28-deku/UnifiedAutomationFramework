# Unified Automation Framework

A scalable UI + API + DB test automation framework built with Java 21.

## Tech Stack
- Java 21 | Selenium 4.38 | TestNG 7.11
- Rest Assured 5.5 | MySQL 8.3
- ExtentReports 5 | Log4j2 | AssertJ

## Test Coverage
- UI: AutomationExercise.com
- API: AutomationExercise REST API
- DB: MySQL validation

## How to Run
mvn test
